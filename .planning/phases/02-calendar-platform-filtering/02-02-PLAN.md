**Modify `filters.tsx`**:

1. Import `SelectPlatform` from `./select.platform`
2. Add `setPlatform` callback (following `setCustomer` pattern):
   ```typescript
   const setPlatform = useCallback(
     (platform: string) => {
       calendar.setFilters({
         startDate: calendar.startDate,
         endDate: calendar.endDate,
         display: calendar.display as 'day' | 'week' | 'month' | 'list',
         customer: calendar.customer,
         platform: platform || null,
       });
     },
     [calendar]
   );
   ```
3. Add `<SelectPlatform>` in the JSX right after `<SelectCustomer>` (line 403):

   ```tsx
     platform={calendar.platform as string}
     onChange={(platform: string) => setPlatform(platform)}
     integrations={calendar.integrations}
   />
   ```
